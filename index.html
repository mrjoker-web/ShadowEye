Network Device Monitor

ShadowEye é a nova adição ao arsenal da Shadow Suite. Monitoriza a tua rede local em tempo real — descobre todos os dispositivos conectados, identifica fabricantes por MAC address, estima o sistema operativo via TTL fingerprinting e alerta-te instantaneamente quando um dispositivo desconhecido entra na rede.

Para quem quer saber exatamente quem está na sua rede.

🔎
ARP Discovery
Scan via ARP — descobre todos os hosts ativos rapidamente
🏭
Vendor Lookup
Identifica o fabricante de cada dispositivo pelo MAC
🖥️
OS Fingerprint
Estimativa de SO via análise de TTL — Linux, Windows, Router
👁️
Watch Mode
Monitorização contínua com alertas de novos dispositivos
🔌
Port Scan
Scan rápido das portas mais comuns por dispositivo
📄
Export JSON/TXT
Exporta resultados para análise posterior
