Shadow Eye.

Construí uma ferramenta open source que monitoriza a tua rede local em tempo real — descobre todos os dispositivos conectados, identifica fabricantes, estima o sistema operativo e alerta-te quando entra um dispositivo desconhecido.

3from scapy.all import ARP, Ether, srp

6# Scan ARP → descobre hosts ativos
7def arp_scan(network: str) → list[dict]:
8 packet = Ether(dst="ff:ff:ff:ff:ff:ff") / ARP(pdst=network)
9 answered, _ = srp(packet, timeout=2, retry=1)
10 return [{"ip": r.psrc, "mac": r.hwsrc} for _, r in answered]
// Features
🔎ARP Scan
Descobre hosts ativos via ARP — rápido e sem falsos negativos
🏭Vendor Lookup
Identifica fabricante por MAC address — offline, sem API
🖥️OS Fingerprint
Estima SO via análise de TTL — Linux, Windows, Network Device
👁️Watch Mode
Monitorização contínua com alerta de novos dispositivos em tempo real
🔌Port Scan
Scan rápido das portas mais comuns por dispositivo (flag -p)
📄Export
Exporta resultados para JSON ou TXT para análise posterior
